SOUNDSCAPE

Visualizador de audio en tiempo real

Para que los gráficos reaccionen al sonido necesitas conceder acceso al micrófono cuando el navegador lo solicite.

Chrome / Edge: Popup en la barra de direcciones 🔒 → Permitir
Firefox: Popup en la parte superior → Permitir
Safari: Menú Safari → Ajustes para este sitio → Mic → Permitir
Archivo local: Si falla, sirve el archivo desde localhost o usa Chrome con --allow-file-access-from-files
